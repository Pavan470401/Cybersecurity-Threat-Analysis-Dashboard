# Cybersecurity Threat Analysis Dashboard

## Project Overview
Developed an interactive Power BI dashboard to analyze cybersecurity threats, financial losses, attack patterns, and industry-wise impact from 2015–2024.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Key Features
- Cyber Attack Trend Analysis
- Financial Loss Analysis
- Attack Type Distribution
- Industry-wise Threat Analysis
- Interactive Filters and Slicers
- KPI Cards and Dynamic Visualizations

## Key Insights
- Analyzed 3,000+ cybersecurity incidents
- Tracked financial losses of 151.48K
- Evaluated impact on 2 Billion affected users
- Identified IT, Banking, and Healthcare as major targets
- Analyzed DDoS, Phishing, Malware, SQL Injection, and Ransomware attacks
